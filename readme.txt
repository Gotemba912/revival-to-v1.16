---★このリソースパックってどういうやつ？？----
┗このリソースパックは、アプデに連れて変わっていってしまったマイクラ統合版のＵＩを
　できるだけ1.16に寄せて戻したＵＩリソースパックです。
　すべてのＵＩが1.16に戻っているわけではないのでご注意を……。

---★このリソースパック使えば1.16に戻せるの？？----
┗いいえ。 あくまでＵＩなどが1.16風に戻るだけで、
　1.16時代にあった機能が使えたりするわけではありません。
　ＲｅｎｄｅｒＤｒａｇｏｎも消せるわけではありません。

---★このリソースパック使えばどんな感じに戻るの？？---
┣今のところこんな感じです:
┃┣・タブをマウスホバーしても緑色にならない。
┃┣・設定画面の「プロフィール」タブを復活。「一般」タブと「アカウント」タブを再統合。
┃┣・設定画面のアクセシビリティ設定のアイコンを元に戻す。
┃┣・設定画面とチャット設定画面の灰色っぽい背景を黒に戻す。
┃┣・日本語翻訳が1.16時代の翻訳を90%再現。 (ブロック名、アイテム名、一部のＵＩ表記。) 
┃┣・日本語翻訳にかつて存在していたガバガバ翻訳を99%再現。
┃┣・スコアボードsidebar、ポーズ画面のUIを戻す。
┃┣・クリムゾン・ゆがんだ木材の音を幹と同じにする。
┃┣・トグルスイッチのOFFの状態のテクスチャが元に戻る。
┃┗・その他いろいろ
┃
┣マイクラのアプデによって動かなくなる可能性があります！
┗また、このリソースパックはマイクラのデフォルトテクスチャにのみ対応しています。
　その他のテクスチャを入れているとうまく適用されないかもしれません！

---★このリソースパックはサーバーとかに適用していい？？----
大歓迎です！自由にサーバー適用したりコピーしてOKです。
また、このリソースパックには1.16バニラのリソースパックから拝借したものがあります。
